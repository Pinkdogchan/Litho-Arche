# Litho-Arche: The Archive of Luminous Memories

リト・アルケー博物誌のWebサイト

## 開発

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## プロジェクト構造

- `app/` - Next.js App Routerのページとレイアウト
- `lore/` - 世界観設定ファイル（Markdown）
- `public/` - 静的ファイル

## デザインコンセプト

- **テーマ**: ダークモード（深淵な夜、宇宙の静寂）
- **アクセントカラー**: 発光（Luminous）をイメージした青系の淡い光
- **トーン**: 幻想的だが、博物誌としての「記録」「標本」という整然とした美しさ
