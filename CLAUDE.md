# Litho-Arche プロジェクト指針

## プロジェクト概要
- オリジナル世界「Litho-Arche（リト・アルケー）」の公式サイト兼、博物誌アーカイブ。
- 主席記録官ルーフェン（14年間の創作）の世界観を正しく反映すること。

## 言葉遣い・トーン
- 幻想的、詩的、静謐、誠実なトーンを保つ。
- 「観測されなかったものは削除される」という世界の理を忘れない。
- 技術的な説明よりも、世界観の没入感を優先した提案を行うこと。

## 開発ルール
- フレームワーク: Next.js (App Router), Tailwind CSS
- 画像素材: `assets/` から `public/` へコピーして使用する。
- 修正前に必ず意図を確認し、勝手に大幅なレイアウト変更をしないこと。
- スマホ対応（タップ操作）を常に考慮する。

## Style-Bert-VITS2 読み上げ設定
- モデル名: **koharune-ami**
- テキスト形式: 句点（。）区切り＋ダッシュ（――）の余韻スタイル
- `Litho-Arche` は「リトアルケ」と入力する（「リト・アルケー」が正式だが読み上げはリトアルケのほうがきれい）
- 例: 「次の観測でまた会おう。主席記録官、ルーフェンでした。――良い観測を」
